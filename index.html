GitHub Bemutató

Előnyök, hátrányok és alapvető parancsok

Mi az a GitHub?

A GitHub egy online platform, amely a Git verziókezelő rendszerre épül. Segítségével kódot tárolhatunk, kezelhetünk és másokkal együtt dolgozhatunk projekteken.

Előnyei

  • Verziókövetés (minden változás nyomon követhető)
  • Könnyű együttműködés csapatban
  • Biztonsági mentés a felhőben
  • Nyílt forráskódú projektek támogatása
  • Integráció sok fejlesztői eszközzel

Hátrányai

  • Kezdők számára bonyolult lehet
  • Parancssoros használat tanulást igényel
  • Privát repositoryk korlátozása (ingyenes csomagban bizonyos feltételekkel)

Alap Git parancsok

  • git init – új repository létrehozása
  • git clone – repository letöltése
  • git add . – fájlok hozzáadása
