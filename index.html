Mi az a GitHub?

A GitHub egy online platform, amely lehetővé teszi a fejlesztők számára, hogy kódot tároljanak, kezeljenek és másokkal együtt dolgozzanak rajta. A Git nevű verziókezelő rendszerre épül.

A verziókezelés azt jelenti, hogy minden változtatás mentésre kerül, így vissza lehet térni korábbi állapotokra.
