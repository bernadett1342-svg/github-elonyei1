GitHub Részletes Bemutató

Mi az, hogyan működik, parancsok és lehetőségek

Mi az a GitHub?

A GitHub egy online platform, ahol projekteket (repository-kat) tárolunk. A Git nevű verziókezelőt használja, ami segít a fájlok változásainak követésében.

Mi az a Repository (repo)?

A repository egy projekt mappája, ami tartalmazza:

  • Forráskódot
  • Dokumentációt
  • Verzióelőzményeket

Ez lehet publikus (mindenki látja) vagy privát.

Hogyan működik?

  • Készítesz egy repository-t
  • Lemásolod (clone) a gépedre
  • Módosítod a fájlokat
  • Mentés (commit)
  • Feltöltés (push)

Előnyei

  • Verziókövetés
  • Csapatmunka támogatása
  • Biztonságos mentés
  • Nyílt forráskód támogatása

Hátrányai

  • Kezdőknek nehéz lehet
  • Sok parancsot kell megtanulni
